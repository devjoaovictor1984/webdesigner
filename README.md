# Web Designer Portfolio

Repositório de projetos web desenvolvidos com foco em layout, responsividade,
experiência do usuário e boas práticas de front-end.

Este repositório funciona como um **mostruário técnico**, reunindo projetos
independentes usados para estudo, prototipação, validação visual e apresentação
a clientes.

---

## 🧩 Tecnologias Utilizadas
- HTML5 semântico
- CSS3 (Flexbox, Grid, animações)
- JavaScript (vanilla)
- Responsividade mobile-first
- Organização modular de assets

---

## 📁 Estrutura do Repositório

Cada pasta representa um projeto independente, com sua própria estrutura de
HTML, CSS e JavaScript.




---

## 🚀 Objetivo dos Projetos
- Criar layouts profissionais e modernos
- Testar ideias rapidamente antes de produção
- Servir como base reutilizável para novos projetos
- Demonstrar domínio prático de front-end

---

## 📌 Observações Importantes
- Os projetos **não compartilham dependências**
- Cada pasta é autocontida
- Alguns projetos possuem vídeos ou imagens demonstrativas
- Não há frameworks JS (intencionalmente)
